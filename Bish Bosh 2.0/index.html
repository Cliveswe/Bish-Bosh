Welcome to Bish Bosh 2.0

Here is a simple example of a number game called "Bish Bosh".

Bish-Bosh lists all the numbers from 1 to ’Max number’, with some exceptions:
If the number is evenly divisible by 3, ‘Bish’ is listed instead of the number.
If the number is evenly divisible by 4, ‘Bosh’ is listed instead of the number.
If the number is evenly divisible by both 3 and 4, ‘Bish-Bosh’ is listed instead of the number.
Enter a number greater than 1.
Enter a number greater than 1 but less than the above max number and not equal to Bosh.
Enter a number greater than 1 but less than the above max number and not equal to Bish.
